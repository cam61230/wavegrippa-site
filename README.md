# wavegrippa.com

## Project: Single-page artist website

**Status:** Prototype v0.1 complete (2026-02-12)

### Design
- Single page with anchor sections: #hero, #music, #mixes, #about, #licensing, #links
- Dark theme, purple/pink/cyan gradients, glassmorphism nav
- British quirky humour / bean theme
- Mobile responsive with hamburger menu

### Brand Assets
- **Logo font:** Uroob Regular (downloaded/installed, not a system font)
- **Logo created in:** GIMP/Glimpse on Ubuntu PC
- **Logo file:** `assets/wavegrippa_logo.png` (white on transparent)
- Font colour and opacity easily adjustable in GIMP
